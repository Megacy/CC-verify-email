[b]Description[/b]
A fixed width theme based on the 1140px grid system and contains a collapsible sidebar where you can add extra stuff such as a twitter feed.

[b]Features[/b]
[list]
[*]JQuery Login Box
[*]Transparent Collapse Images
[*]XenForo Styled User Panel
[*]Sexy Breadcrumbs
[*]Collapsible Sidebar
[*]CSS3 Buttons
[*]Simple, Elegant Postbit
[*]IPB Style Stats
[*]Tool Tip Descriptions
[*]Sexy Icons in User Panel
[/list]

[b]Installation[/b]
1. Upload the /themes/ folder and all its contents to your root directory.
2. Upload the folder labeled 'narrow' to jscripts/editor_themes 
3. Install the .xml file in administrative control panel.
4. Enjoy!
